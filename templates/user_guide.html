Home Page - Choosing the project

After login the user will land in the home page. There, the user will be asked to select the Company in the dropdown menu and click on Submit button;

In the “Instructions for the input templates” section there is the “CMDB & OneITSM Templates button. By clicking on it, the user will land on the SharePoint folder where are all the Templates files the user needs to start a data collection.
